# Homework-Uploads

VALEO × AdAstra Kurumsal Yapay Zekâ Yetkinlik Programı — Ödev #1 teslim deposu.

## Nasıl yüklenir?
1. Üstte **Add file → Upload files** butonuna tıklayın.
2. Kendi adınızla bir klasör oluşturun (ör. `Mehmet-Yilmaz/`) ve dosyalarınızı bu klasörün içine koyun.
3. Klasördeki tüm dosyaları sürükleyip bırakın (alt klasörler korunur).
4. Altta kısa bir not yazıp **Commit changes**'e basın.

Klasör yapısı, teslim şartları ve değerlendirme kriterleri ödev dosyasında ayrıntılı olarak yer almaktadır.

Teslim son tarihi: **18.06.2026 – 23:59**
